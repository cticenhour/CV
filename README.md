# Simple CV/Resume

This repository houses the current curriculum vitae (CV) TeX source files for
[@cticenhour](https://github.com/cticenhour).

## Building

Compilation into PDF format is easily done by running `make` within the repository
root directory. Note that `latexmk` is required to build, which should be available
with many LaTeX distributions (like TeX Live).

> [!TIP]
> This document workflow can take advantage of the `latexmk` "preview continuously" workflow to
> allow for on-the-fly updates to the file as dependencies are being edited. To use this, simply
> build with `make watch` instead of the standard build command.
